Detailed Description

template<typename T, StorageOrder S = StorageOrder::Rows>
class algebra::Matrix< T, S >

Matrix template class that considers row ordered matrices as its default case. It can deal with both uncompressed matrices and with CSR matrices.

Template Parameters
Ttype of the Matrix object elements
SStorageOrder of the matrix (either StorageOrder::Rows or StorageOder::Columns)
